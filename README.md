# Puzzle & Monsters

`Puzzle & Monsters` は、ジェムを入れ替えて戦うパズルバトルゲームです。プレイヤーはパーティを率いて、次々に現れるモンスターと戦います。

## ゲームの流れ

各ターンで 2 つのジェムを入れ替え、同じ属性のジェムをそろえると行動が発動します。属性ジェムがそろうと仲間モンスターが攻撃し、`LIFE` のジェムがそろうとパーティを回復できます。

敵との戦闘は 1 体ずつ進み、最後まで勝ち抜けばゲームクリアです。パーティの HP がなくなるとゲームオーバーになります。

## 操作

- プレイヤー名を入力して開始します。
- 交換したい 2 つの位置を 2 文字で入力します。
- 入力できる範囲は `A` から `N` までです。
- 同じ文字を 2 つ続けて入力することはできません。

## 補足

敵を倒すたびに先へ進み、最終的にドラゴンを倒すとクリアです。
